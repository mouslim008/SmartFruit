# SmartFruit 🍎

An intelligent Flutter application that uses AI to identify fruits through camera or photos and provides detailed information through a voice assistant.

## Features

### 🎯 Core Features
- **AI Fruit Classification**: Uses TensorFlow Lite to identify fruits from images
- **Voice Assistant**: Ask questions about fruits and get spoken responses
- **Multi-platform**: Works on Android, iOS, Web, Windows, and macOS
- **Firebase Integration**: User authentication and data storage
- **Real-time Processing**: Instant fruit identification

### 📱 Screens
1. **Login/Register**: Secure user authentication
2. **Home**: Main dashboard with feature access
3. **Classification**: Camera/gallery fruit identification
4. **Voice Assistant**: Interactive fruit Q&A

### 🔧 Technical Features
- TensorFlow Lite for on-device ML
- Firebase Auth & Firestore
- Speech-to-Text & Text-to-Speech
- OpenAI GPT integration
- Responsive UI design

## Prerequisites

- Flutter 3.0+
- Dart 3.0+
- Firebase account
- OpenAI API key (optional)
